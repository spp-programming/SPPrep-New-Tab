What's New

Last updated as of version 3.1. This page reflects all changes made up to the currently installed version, but not newer ones.
If you're looking for an update to a newer version, you should wait until your device receives the update automatically. Unfortunately, we can't do anything to expedite this process.

Version 3.1 (Latest, installed version)

This is a minor release because it does not contain any new major features.

✅ Fixed small bugs:

  • 🥞 Buttons now stack up more neatly as needed
  • ↔️ Added padding to this changelog page.

🧑‍💻 Behind the scenes:

  • ✨ More cleanup
  • 📝 Fonts are now compressed using woff2.

Version 3.0 (Previous version)

This is a major release.

🚨 Fixed major bugs:

  • 📅 Letter day lookup now shows the correct day, using a new secret sauce logic (NoLetter-DAY, we won't miss you)
  • 🔔 Bell schedule button now works as intended
  • 🆕 Bell schedule itself has been updated

✅ Fixed small bugs:

  • 🕒 Time is now accurate down to the millisecond
  • 📬 Gmail icon has been updated
  • 🔗 Gmail, Drive, and School Calendar links are more generic (better behavior with multiple accounts)
  • ⚡️ Minimized flash of unstyled content
  • 📝 Better grammar and spelling

🧑‍💻 Behind the scenes:

  • ✨ Refactored, more maintainable codebase
  • 🔌 No more remote fonts! Network access is now only used for letter day lookup.

🎉 New features:

  • 📜 This changelog page and its corresponding button
  • 🗓️ Added Google Calendar button (separate from school calendar)
  • ♿️ Accessibility improvements
  • ❄️ New Fall/Winter background
